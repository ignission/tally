## 概要

決定論的ルールで孤立ファイル・コード重複・ADR強制ルールの整合性を検出し、リファクタリング候補をリストアップする。
削除・修正の判断はユーザーが行う（自動修正しない）。

## 手順

1. 検出スクリプトを実行する

```bash
bash "$CLAUDE_PROJECT_DIR/.claude/scripts/garbage-collect.sh" "$CLAUDE_PROJECT_DIR"
```

2. 結果を確認し、各候補について以下を調査する:
   - 孤立ファイル: 本当に参照がないか（動的importやマクロ経由の参照がないか）
   - コード重複: 共通化すべきか（意図的な重複ではないか）

3. 結果をテーブル形式でユーザーに報告する

### 孤立ファイル

| #   | ファイル | 種別 | 参照状況 | 削除推奨 |
| --- | -------- | ---- | -------- | -------- |

### コード重複（リファクタリング候補）

| #   | ファイルA | ファイルB | 重複行数 | 共通化方針案 |
| --- | --------- | --------- | -------- | ------------ |

### ADR強制ルール（archgate）

| #   | ADR | 参照ルール | 状態 |
| --- | --- | ---------- | ---- |

- 壊れたポインタ: ADRが参照するルールファイルが存在しない

4. ユーザーの判断を待つ（勝手に削除・修正しない）
